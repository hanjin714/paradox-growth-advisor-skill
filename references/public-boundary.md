- Common judgment frameworks.
- First-step actions.
- Public examples.
- Basic lead table fields.
- Simple A/B/C lead grading.
- General Agent / workflow / Codex choice logic.
- Basic prompts for self-diagnosis and first records.

## Public Wording Rule

Use:

```text
公开版讲判断框架和第一步动作。
具体行业字段、完整流程和自动化实施，需要回到真实业务里判断。
```

Avoid:

```text
我把完整方法都免费给你。
照着做就能成交。
全自动帮你把客户转化。
```

## Compliance Rule

For public videos or platform-facing text, avoid direct off-platform traffic wording such as “加微信、进群、私域、保证成交、快速变现、无人值守成交.”

Use safer wording:

```text
平台允许的方式领取基础模板。
合规承接。
减少漏线索。
跑通小闭环。
AI 内容实验。
```
