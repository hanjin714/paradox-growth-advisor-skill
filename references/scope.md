# Scope

Use this file when the Skill needs audience-facing scope, platform-safe wording, or a clean explanation of what the first-round method focuses on.

## Core Output Range

The Skill focuses on:

- Common judgment frameworks.
- First-step actions.
- Scenario-level examples.
- Basic lead table fields.
- Simple A/B/C lead grading.
- General Agent / workflow / Codex choice logic.
- Starter prompts for self-diagnosis and first records.
- 7-day or 30-day execution plans that a small team can run.

## Depth Rule

Keep outputs close to the first runnable business loop:

```text
业务循环 → 最大漏点 → 第一张表 → A/B/C 分层 → 7 天动作 → 复盘问题 → 工具选择
```

Use scenario-level examples instead of over-complex systems.

When a task becomes too broad, pull it back to one question:

```text
这一步到底要让用户看见、开口、记录、跟进、复盘，还是复购？
```

## Platform-Safe Wording

For public videos or platform-facing text, prefer:

```text
领取基础模板。
合规承接。
减少漏线索。
跑通小闭环。
AI 内容实验。
线索记录。
下一步动作。
复盘表。
```

Avoid hype-driven wording that sounds like guaranteed results.

Use this frame:

```text
先把第一轮业务动作跑清楚。
再根据真实记录决定下一轮加深内容、线索还是工具。
```
